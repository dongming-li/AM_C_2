Description of folders in project

Config:
Contains config files. Most notably for Apache and PHP, as I know they require config files that can be fairly extensive.

Examples:
Contains examples and other useful documentation.

PC_Client:
Contains all C#/VB files, and the Visual Studio project for the PC Client application

Server:
Contains all Java and other source/required files for the Java server (core application)

Web_Client:
Contains all components of the application web interface. (PHP, JavaScript, HTML, CSS, etc. as well as Apache PHP config files)
